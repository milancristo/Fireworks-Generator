Interactive Fireworks Generator using HTML5 Canvas. This application will function as a single file, featuring a full-screen display and a control panel that allows you to customize the physics, colors, and intensity of the fireworks in real-time.

Key Features
Interactive Canvas: Click or tap anywhere on the screen to launch a firework at that specific spot.

Customization Panel: Tweak gravity, particle count, explosion size, and wind.

Color Modes: Choose between specific colors, random mixes, or "complementary" two-tone styles.

Auto-Fire Mode: Sit back and watch a generated show based on your settings.
